# Funding Targets

Grants and funding programmes aligned with Iskander. Ordered by fit and urgency.

## Priority 1: NLnet NGI Zero Commons Fund

- **URL**: https://nlnet.nl/commonsfund/
- **Apply**: https://nlnet.nl/propose/
- **Amount**: EUR 5,000 — 50,000 (scalable if proven)
- **Deadline**: June 1, 2026 (IMMINENT — next call)
- **Eligibility**: Anyone. Must be FOSS. All outputs open access.
- **Fit**: Excellent. NLnet funds "breakthrough open internet projects" — federated infrastructure, P2P, privacy, libre software. Iskander hits every theme.
- **What to request funding for**:
  - K3s Helm chart development and testing (Phase C installer)
  - MACI ZK circuit compilation and trusted setup tooling
  - Headscale federation protocol implementation
  - Hosting infrastructure for demo/testing cluster
  - Security audit of the self-hosted stack
- **Key messaging**: Iskander is federated internet commons infrastructure. Not blockchain hype — cooperative tools that use cryptography only where nothing else works. AGPL-3.0 licensed. Designed for communities that can't afford enterprise tools.
- **Previous NLnet grants in similar space**: They've funded Matrix/Dendrite, Loomio-adjacent governance tools, ZK infrastructure. Strong precedent.

## Priority 2: Interledger Foundation

- **URL**: https://interledger.org/
- **Amount**: Varies (DisCO received a grant previously)
- **Deadline**: Rolling / periodic calls
- **Fit**: Good. Inter-cooperative payments, cFIAT token settlement, escrow contracts.
- **What to request funding for**:
  - Inter-cooperative payment protocol (cFIAT)
  - Escrow smart contract development
  - Integration with Interledger protocol for cross-cooperative settlement
- **Key messaging**: Cooperative-to-cooperative payments without intermediary extraction. 1:1 fiat-backed tokens, not speculation.

## Priority 3: Power to Change — Community Tech Discovery Fund

- **URL**: https://powertochange.org.uk/our-work/our-programmes/community-tech
- **Amount**: GBP 10,000 (Discovery Fund)
- **Deadline**: Rolling (check website for current status)
- **Eligibility**: Community businesses in England. Focused on community of place. Prioritises marginalised communities.
- **Fit**: Good if framed as community tech for housing/worker co-ops.
- **What to request funding for**:
  - Feasibility study: deploying Iskander in a real UK housing cooperative
  - User research with co-op members on governance tool needs
- **Key messaging**: Community-controlled tech infrastructure. Self-hosted, no corporate dependency. Designed for democratic organisations.
- **Note**: Power to Change is shifting from direct grants to advocacy. Check current programme status before applying.

## Priority 4: Cooperative Development Foundation (CDF)

- **Deadline**: Annual (concept notes typically Feb)

## Priority 5: National Lottery Community Fund

- **Fit**: Moderate. Broad community focus, not tech-specific. Would need strong community benefit narrative.

## Priority 6: Innovate UK / UKRI

- **URL**: https://apply-for-innovation-funding.service.gov.uk/
- **Amount**: GBP varies by competition
- **Deadline**: Competition-based

## Open Source Sustainability Directory

- **URL**: https://www.oss.fund/
- **What it is**: Catalogue of funding sources for open source projects.
- **Use**: Cross-reference for additional opportunities. Includes GitHub Sponsors, Open Collective, Sovereign Tech Fund, etc.

## Other Avenues

- **GitHub Sponsors**: Set up once repo has traction. Recurring funding from individual supporters.
- **Open Collective**: Transparent budgeting for open source projects. Good for small donations.
- **Sovereign Tech Fund (Germany)**: Funds critical open digital infrastructure. Larger grants.
- **Mozilla Foundation**: Periodic grants for privacy and internet health projects.
- **Ford Foundation**: Technology in the Public Interest programme. Larger, more competitive.
