# Assembly Guides

Step-by-step instructions for building, flashing, and activating Iskander Hearth nodes.

## Guides

| File | Who It's For | What It Covers |
|---|---|---|
| [`building_the_commons_node.md`](./building_the_commons_node.md) | First-time builders | Physical assembly of the Tier 2 Commons Node from BOM components |
| [`flashing_iskander.md`](./flashing_iskander.md) | Anyone setting up the OS | Flashing Iskander OS, the Ad-Hoc setup flow, UPS and power sensor wiring |

## Before You Start

1. **Read the BOM first.** Confirm you have every component in [`boms/tier2_commons.csv`](../boms/tier2_commons.csv).
2. **Read the procurement checklist** in [`supply_chain/procurement_guidelines.md`](../supply_chain/procurement_guidelines.md) — verify all refurbished components before assembly.
3. **Log your components** in the Material Passport before opening any boxes.
4. These guides assume **zero prior PC building experience**. If you have built a PC before, skim the safety section and skip to the step that matches where you are.

## Tools Required (for both guides)

- Phillips #2 screwdriver (magnetic tip strongly preferred)
- Phillips #1 screwdriver (for small SSD and TPM module screws)
- Anti-static wrist strap + grounding mat (or work on a bare wooden table, never carpet)
- Needle-nose pliers (for standoffs)
- USB drive, 32GB minimum (for Iskander OS installer)
- A second computer (to download and write the Iskander OS image)
- Cable ties or velcro straps (for cable management)
- Thermal paste (included in BOM as T2 — or use pre-applied paste on cooler if present)
- Small flashlight or headlamp (chassis interiors are dark)
